Hi, I just wanted to mention that I changed the format of the code
for this box. It was originally a letter-number cypher, which I
didn't like. I've changed it to a typical cryptogram with letter
substitutions. I like this format better. I think it's a little
easier to see the letter patterns and solve the puzzle. The actual
clues to the box did not change, so if you already figured it out
using the numbers, no worries. If you looked at it before and the
number cypher was too hard to "see", check it out now. It's easier
on the eyes.
