I have had the same thing happen to me but I have far less Spam
coming in (yikes! 500 messages!?) so I don't think contact the
placer messages have slipped past me.

When I read your note, I thought of a possible solution. This may or
may not work, but I have just implement it myself.

Try adding the user: notify at letterboxing dot org
(notify@letterboxing.org) to your email address book. I think those
spamguard type programs have a feature that scans the address book
before sending an incoming message to the bulk/junk mail box.

If you try it, let me know. Maybe we can test each other's mailbox
by using the contact placer feature at letterboxing.org (grin)

Enjoy!

From: Doodle & Deedle Bug (doodle_n_deedle@yahoo.com) | Date: 2004-10-14 07:03:11 UTC-07:00
And if that doesn't work, you might be able to set up a filter so that all messages that arrive with a certain email address or word in the subject, etc., get placed into your main inbox rather than the spam folder. We've got our yahoo email account set up to sort the messages we get from our letterboxing talk lists into specific folders so we're not reading messages one right after the other & thinking, "Which group does this have to do with, now?" I'm pretty sure an SBC Yahoo account has the same options for setting up filters, etc., as a regular free yahoo account.

Doodle.
