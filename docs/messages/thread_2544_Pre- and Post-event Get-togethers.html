There seems to be a bit of confusion about the purpose of the
database that speedsquare created.

What we're trying to do is find out how many people might be coming
to a get-together at the Delafield Brewhaus on Friday night (Pre-
event) and/or to a get-together Saturday night at Marty's Pizza (Post-
event) so we can try to reserve some space. Both of these occasions
will allow more time for letterbox chit-chat and exchanges.

If you might come to either of these, use the database to record what
time you might get there. These would be evening times.

There is no need to make an entry if you are just coming to the main
event during the day.

Hope this helps!
