Only one week to the gathering. WOOT!

On a side note-

It would seem that I'm not the only one placing very colorful stamps
in the midwest these days. I would suggest, if you haven't done so
already, that people purchase a package of multiable colored inking
pens to bring to the gathering. Michael's usually has a coupon in
the Sunday paper. I'd suggest using that. If you are going to
stick with a single ink pad, please be sure to bring "wipees", of
some sort, to clean off the stamps (as usual)when you are done.
That way those of use that use inking pens won't have them ruined.
Thanks!
