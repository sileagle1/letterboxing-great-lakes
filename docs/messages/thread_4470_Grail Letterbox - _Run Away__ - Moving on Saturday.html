Greetings!

The McHenry County Conservation District in Northern Illinois has put
some guidelines in place regarding the planting of letterboxes and
geocaches. One of these is to register all boxes, and gain approval
for their locations.

I recently followed the process for attempting to register the 'Run
Away!' box from the Monty Python and the Holy Grail Series.
Unfortunately, the spot in which the box has resided for the last
couple of years is considered to be in an area that is set aside for
research, and is not an allowable place for boxes. I received word
that the box needs to be moved, or it will be collected and discarded
at the beginning of October.

That said, I am going to retrieve the box tomorrow morning (9/29), and
will begin working on getting a new, approved location and drafting a
revised set of clues. My intention is to find an approved location in
the same park.

If you have any questions regarding the box, please don't hesitate to
ask. As soon as it is replanted, I will let everyone know so that
anyone questing for that box can resume the hunt!

Best Regards,
