From: Atom118 (Aatom118@gmail.com) | Date: 2011-06-01 08:51:27 UTC-05:00
Today we launched the 'Search for the Key.' In order to find the key,
you need to find the other 18 boxes in the series. As you find each box,
RECORD THE CODE WRITTEN ON THE 2nd PAGE OF EACH BOX'S LOGBOOK. THESE
CODES ARE CASE SENSITIVE, SO MAKE SURE YOU COPY THEM EXACTLY. EACH CODE
HAS A NUMBER (1-18) AND THEN A 4 CHARACTER CODE. By DECIPHERING THE
CODES, you should be able to find this box. Thanks to Hartx6 for carving
the stamp.

Clues for all of the boxes should be posted today, and they will be
linked to the clue for the final box.

See http://www.atlasquest.com/boxes/clue/?gBoxId=172609 or
http://www.letterboxing.org/BoxView.php?boxnum=58084&boxname=Search_for_the_Key:_Key_to_the_City

The Story:

At a recent event, the following was overheard spoken by Mayor Daley to
Rahm Emanuel, I wish you the best of luck, and I had hoped to give you
the KEY I inherited from Eugene.

While a small side note, it piqued the interest of several of the press
corps. Could this be the fabled Key to the city? It was rumored to have
been presented to Mayor Ogden by a local civic group back in 1838.
Although the actual key had never been seen, it is said to unlock all
the mysteries of the city of Chicago and Lake Michigan. Its existence
had been, long ago, dismissed as an urban legend, and even among the
true believers, it was thought to have vanished decades ago. Did Mayor
Daley actually once possess the key? Maybe, he was hazing the new guy
and just perpetuating a known Chicago myth.

If you are as curious as I, you will need to search the region to find
clues to help you find the whereabouts of the lost Key. Lucky for you,
Ive done a bit of homework, and Ive uncovered some information that
you will find helpful. In doing so, you might decipher the code that has
kept the Key to the city hidden for nearly 200 years. You will have to
visit some old haunts, past notable figures, and find a few relics of
Chicago history. Now, a few of my leads are purely speculative, but a
few of them are much more likely. So to complete this journey, youll
have to visit each lead.

GOOD LUCK!
