Glad you finally found the notorious "Butt Box" aka Masquerade #1!
It took a long time (four months) to write those clues, so it's
gratifying to see them leading 'boxers to victory (finally!).

What a bummer about Herrick Lake, though. That was the first Lb
stamp I ever carved. Oh, well.... Rest assured I will make a note
of its absence on the LbNA site and begin carving a new (and
hopefully improved) stamp in the near future.

As for the Terrified Tree.... Did you happen to find an extra set

bonus box clues....

A placer's work is never done (thank goodness!)
