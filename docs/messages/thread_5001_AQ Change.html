Here's a message Otis' Friends listed on the Mid-Atlantic board:

================================
Those of you with clues on Atlas Quest will want to know that Green

>I'm surprised that Ryan set the default to ALLOW comments, since
>there has been a lot of concern about what type of "spoilers", etc.

Lots of people agreed with this. Ryan just changed it so that the
default is "moderate comments" so the planter/owner has control over
what gets posted. You can go into your preferences and select
options to automatically post comments (you can delete anything you
don't like later) or you can select no comments so you don't have to
approve anything and nothing can be posted.

I like the idea of comments, but I did switch my option to moderate
before he set that as the default. It could be pretty handy for
boxes that are outside of a planters stomping grounds, but need
maintenance or for planters that go MIA. It'd be helpful to the next
person to know that clues may need to be updated or maintenance is needed.

hx6,
There's an option for you as a finder leaving the comment to select
"public" or "private". So, if you just want the placer to see it,
you can check private.

If you as a placer have a bunch of "nice stamp" or "thanks" comments,
you also have the option to delete those posts. Actually, he's
changed it now so that it defaults to moderated. So, you actually
have to approve all comments before they're posted to your boxes. If
you don't want to moderate them, you can go to "preferences" and then
miscellaneous to change your settings to display all (with option to
delete later) or display none.
