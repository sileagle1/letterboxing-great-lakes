From: Mosey (PonyExpressMail@comcast.net) | Date: 2005-08-19 19:51:39 UTC-05:00
I found out just a bit ago how I goofed. Yes, the box is still there. But
because I changed my status on it to found, it disappeared from *my* screen
when I open the LbNA northern Illinois boxes page. I hadn't been doing much
of the "status" stuff before so I didn't realize it was taking the boxes off
the page. I don't really want the page changed that way when I open it, so
I changed all my "status"es back to "none." And now the box appears back on
the main page again.

I don't like the feature then because if boxes change, I'm not likely to
notice it. For example, "Peck Farm" in Kane County. I just noticed that
the placer not only changed the locations of the two boxes I've already
found but has added a third box. And I never noticed the changes.

So I decided not to keep track on LbNA and changed everything back. And now
I see *all* the boxes again on one page.

~~ Mosey ~~

RE: ATTN Flash RE: [LbGLK] Another very quick REPORT: Cook County, IL

From: Debbie Kotlarek (kotlarek@wi.rr.com) | Date: 2005-08-19 20:57:18 UTC-05:00
You can define your settings to show all the boxes, even if you have logged
them as found. All you need to do is be sure you have the following set in
Member Services:

Hide Found - hide boxes that you've found or set to ignore; currently
displaying.

The key is "currently displaying". That means you want to display them even
if you logged them as found.

Wisconsin Hiker
