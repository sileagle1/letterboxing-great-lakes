Holy Grail Series Update

From: duncan53211 (duncan53211@yahoo.com) | Date: 2008-07-05 20:45:21 UTC
I've processed all the information I've received about the Holy Grail
boxes and am in the process of updating the clues with any info that
has been passed on to me.

I have uploaded a pdf copy of my Excel spreadsheet which lists each
box, planter, location, status, and extra info about the condition of
the box. You will see that there are a number of issues that need to
be addressed. If you are responsible for a box, please do me the
favor of checking on it this month (if you haven't already done so
recently) and fix anything (including clues) needed. If you are
unable to do so, please request for someone else to do it on the GL
site. Be sure to let me know of the status.

If you were the carver for a missing or possibly missing box, please
work it out with the planter as to how you will go about replacing the
box. Some carvers are no longer letterboxing, so we may need some
help recarving stamps from the GL group as well.

We still need adopters for Howling Coyote's and Armadillo Jo's boxes.
Contact me if you are interested. Since Team Taco and Gooble
Gobble have taken a break from letterboxing, we also need volunteers
to check up on their boxes. Last I heard was that the Migrating
Coconuts were in GG's Sir Robin's Minstrels box, but that info was
from over a year ago.

Thanks!
Dragon
