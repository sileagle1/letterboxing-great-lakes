Smart Phone

From: Norse Gnome (norsegnome@yahoo.com) | Date: 2011-12-28 18:56:42 UTC-08:00
I have the Motorola Droid 2 thru Verizon.
Love it; especially the Box Radar Ap; we have totally used it on the road, spur of the moment to letterbox. We now keep a spare signature stamp and index cards in the glove box at all times.
I have the 3GMobile set up on the phone so we can use it as wi-fi forour laptops.
We use this a lot on trips--in the car riding down the highway, in restaurantsand found it to be more dependable than hotel wi-fi. We now carry a cheap printer so we can print tickets, confirmations, maps, etcon the road.
TheDroid camera doesn't replace my Panasonic Lumix, but on a long hike it sure weighs a lot less and the compass and mapping aps are sweet.
