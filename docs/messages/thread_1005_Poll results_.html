The winner is... "Hometown Landmarks"

The point of the Great Lakes Letterbox Exchange is to get the group
working on a joint project. It is my experience that group projects
like this tend to be more successful and more easily recognized if
there is a central theme connecting it all together. Because I
wanted to make this a group effort, and not just my project I put the
theme up for suggestion and vote instead of just mandating one.
Understandably in a group with this many people, not everyone is
going to want the same theme, and the fairest way to do this is to
put it up for vote. I would hope that the fact that your favorite
theme was not chosen will not keep anyone from playing, after all we
are all adults. It is more about working on a joint project than
about the specific theme that was chosen.

get in on the fun.
