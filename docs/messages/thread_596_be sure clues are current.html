Just a reminder to be sure when you are ready to hunt a box that you
have current clues. I heard from a searcher about one of my boxes
and he asked what a particular clue meant. It was where the box was
supposed to be, but he had found it in a different place. I had
changed the clues because the box had been damaged in the original
hiding spot. Well, he found the box in the original place. So I
assume someone else had found the box with the old clues and had put
it back in the original place...which was now the wrong place.
Whew, does any of this make sense? :-) Anyhow..bottom thought is be
sure the clues you have are up to date.

Grandvilla
