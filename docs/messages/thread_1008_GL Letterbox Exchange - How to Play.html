GL Letterbox Exchange - How to Play

From: SpringChick (springchick1219@comcast.net) | Date: 2005-02-04 17:56:35 UTC

Okay, the theme has been decided -- Hometown Landmarks! The
letterbox you create should be loosely tied to a landmark in the
city/county/state where you live. It would be nice if you include
brief information about the landmark inside the logbook, but this is
not necessary. This is a great theme and should produce a really
cool series of boxes!

Specific details of how the exchange will work can be found in
message #2680. If you would like to be a part of this group project,
please e-mail me offlist with the following information:

Name
Trail Name
Mailing Address (this will be given only to the person who will be
sending you a box)
E-Mail Address

The original date to sign-up by was 2/5, but since we had the tie-
breaker voting, I will change that to 2/19. Once the sign-ups are
in, I will make random assignments as to whom you will be making a
box for and will let everyone know by 2/22. The person whom you will
be receiving a box from will remain a surprise until your box arrives
in the mail.

Boxes will need to be mailed to your recipient by 4/15/05. If you
only have time for one letterbox between now and then -- please
consider doing a box for the Great Lakes Gathering first as it is not
my intention to compete for boxes with the that event.

If you have any questions, please do not hesitate to post to the list
or e-mail me privately. Thanks for playing!

SpringChick
