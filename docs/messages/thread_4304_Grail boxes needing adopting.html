Re: Grail boxes needing adopting

From: duncan53211 (duncan53211@yahoo.com) | Date: 2007-07-29 16:52:04 UTC
Adopting a grail box is really no different than having your own box
that you planted. Know where it is, check up on it periodically and
maintain it, check on it if someone reports it missing, make sure the
clues still fit according to environmental changes. About the only
difference is that if it does go missing, the original carver would
probably be the one to re-carve it, if not put a whole new box
together for you.

As to how often a box should be checked on, that is up to the placer.
It also depends on how busy you are, how far away it is, etc. I now
have over 200 boxes and there is no way I can check on them all the
time, so I mainly rely on finders reporting and fixing problems. But
if I'm traveling past the spot where i have a box, I usually stop by
to check it. The GL group also has April as its "check your boxes"
month to encourage getting the boxes ready for the summer season.

Dragon
