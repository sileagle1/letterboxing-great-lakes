Hey Team 5th Wheelers...

The Info Sheet can be found in the files section of this group.

Go to the group home page online (http://groups.yahoo.com/group/LbGLK/)
Click on FILES in the left-hand menu section
Double-click on the folder called !Letterbox Includes
Scroll down to the Info Sheet
It is there in both .doc (MS-Word) format and .pdf (Adobe Acrobat)

Let me know if you have any problems.

Have a wonderful trip!
