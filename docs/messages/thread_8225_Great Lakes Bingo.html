Quest Scouts BINGO started today which reminded me that Great Lakes Bingo will start January 1, 2016!  


I know it is early to start thinking about it, but if you have suggestions for any new twists for the next year of Bingo, let me know.  It seemed like "Biggest Loser" was popular last year, so we'll be keeping that.


ALSO:  If you plan to play GL 2016 BINGO , you can bring your Bingo booty to the Fall Folk Festival event in Wisconsin on OCT 17 or the Fall Into Fun event in Illinois on NOV 7.  That will save mailing costs.  We're not sure yet if we will be able to attend the Michigan event on OCT 24.  If we do, I'll be able to collect booty then also.
