From: baliddle (baliddle@sbcglobal.net) | Date: 2007-06-13 18:08:11 UTC-04:00
Hey there!

I had a box I planted for one day only at the GL event a couple weeks
ago. I'm planning on planting it in a permanent location in MI this weekend.

If it were you, would you just use the same info on LbNA and AQ and
change the location? Or should I archive those listings and start it
as a new listing in case anyone who found it before wants to find it
in it's new home? A very few people logged into it in OH, and they
probably wouldn't be finding it here anyway. But, if they did, I
think it wouldn't log as a new find if I left it as it is now.

For those of you that have done this, how'd you handle it?



Beth

From: Wisconsin Hiker (kotlarek@wi.rr.com) | Date: 2007-06-14 00:20:20 UTC
The main thing I would appreciate is that you note in the clues that
the stamp/box was originally placed at xxx gathering. That way those
that already found it would know that it is the same box. In the
past, we've done a fairly long walk to find a box, only to discover
it was the same box that had been hidden before at an event we
attended, and that we had already previously found it.

Wisconsin Hiker
