This sounds so childish, but in the spirit of not
having duplicates, here goes.


Ahem, Ahem,

I, Gooble Gobble, hereby do proclaim the
following images to be the sole right of
aforementioned person, me, to be exclusive unto my
carving rights set forth in said document, that would
be this one, which you've just read.
(That was fun to write)

1. Minnesota State Seal
2. Minnesota State Flower (Pink and White Lady's
Slipper)
3. Minnesota State Insect (Monarch Butterfly)
4. Minnesota State Quarter
5. Minnesota State Fish (Walleye)
6. Minnesota State Mushroom (Morel)

I hope no one feels I'm "hogging" images, but I want
to carve alot for this gathering since we will be
hosting all of you fine people. I have a huge stack of
carving medium to use and what better thing to use it
for.


Gooble Gobble

P.S. I can't wait to see the wood chipper stamp.
