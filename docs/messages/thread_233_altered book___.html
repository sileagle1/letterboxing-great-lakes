In regards to the question as to whether this has been done
before... There are "letterbooks" that have been placed in
libraries which utilize hollowed out real books for the container,
as Richard had mentioned. I have not seen any of these so I'm not
sure if they just remove the entire inner hunk of pages or cut
individual niches for the stamp and logbook, or if part of the
original book is utilized for the journal, etc. I also don't know
in what way they are painted, decorated, etc. By precise definition
of "altered book" adornment is not mandatory, rather simply cutting
a niche technically qualifies as "altering." My guess is that your
book is more artistic than most of these, more closely fitting the
idea we hold as altered book artists.

Either way, it is a very cool concept for an indoor letterbox and I
would like to see more of these types of boxes as they really allow
for some creativity.

If anyone is interested, I believe the very first "letterbook" was
the Kimball Library box placed by Tom Cooch, a great "old-time"
letterboxer. The box is in Vermont, but there is an on-line version
(in this respect it probably qualifies also for the first virtual
letterbox). Even though Tom is not so actively involved in the
letterboxing community as he previously was, it does appear that he
still keeps up with this box.

http://www.sover.net/~tcooch/kimball.htm
