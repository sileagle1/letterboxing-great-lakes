What a great tip to share, thanks!

Here's one from me:

For my stamps, I use that darn crumby stuff because its carves
sooooo easily. There have been times that small, but important bits
have broken off; especially when I'm trying for a really fine line.
I've repaired those void areas using the matte gel medium that I use
for mounting by building up the area in layers. There are several
other products that could be used in the same manner, for the same
purpose.

Enjoy!
---PJ
aka Bindle Babe
