Wirth Park box

From: Wisconsin Hiker (kotlarek@wi.rr.com) | Date: 2010-04-02 03:31:48 UTC
Well I finally replaced my Wirth Park box! It was the first box I ever planted (in OCT 2003), but it was destroyed by an animal in 2008. I went to replace it last year, but since one of the trails was under construction, I waited. Today was such a glorious day that I headed over to the park, the trail work was mostly complete and I was able to find a new spot for the box. Although almost all of the Milwaukee area boxers found this box long ago, the replacement is now available for any newer folks.
