RE: [LbGLK] Re: Migrating Coconuts???

From: Debbie Kotlarek (kotlarek@wi.rr.com) | Date: 2006-09-21 19:47:23 UTC-05:00
This HH is always supposed to be reported to Dragon and this message board
and only moved to Grail boxes. Here is the info from the Grail site and I
believe this same info is with the HH:

Migrating Coconuts Hitchhiker


Created by: Papeseed
Current Location: ~currently in flight~


Clues:

This hitchhiker is to be moved only from grail box to grail box. Its
current location will always be noted above. Do not place it in a regular
letterbox! If you will not be going to another grail box soon, leave this
HH where you found it - do not "sit" on this HH. We would like this HH to
travel to all the states of the GL region so everyone has a chance to get
it, so if you can help out please move it along. Report all information
about the location and status of this HH to The Dragon and the GL and/or
LBNA message boards. Thanks!
