Ok, who's up for hosting one? We don't have to make it this year.

and start talking about it! If each state would host one, (I always
feel sorry for MN, it's really far where ever the GL gathering is)
at different times troughout the summer, that way if you really
wanted to go to, lets say MI, but its not being held at the right
time for you, than you can go to another state's gathering close by.
I am willing to volunteer for MI State Annual Gathering mid summer.
Each year it will be hosted by a different host at a different
location, so as not to be partial to on side of the state or the
other. It will be at Maybury Park (hopefully). The theme will
be "Saturday Morrning Cartoons", "Mythology", or "Where No Man Has
Gone Before" (if I can get permission from a co-conspirator). There
will be a vote on the date, theme, and if I can think of an
alternate park. Anyone else want to step up to the plate?
