So, which state is best-represented

From: Doodle & Deedle Bug (doodle_n_deedle@yahoo.com) | Date: 2004-10-08 13:23:49 UTC-07:00
Since it's Friday afternoon and I'm here at work, I decided, in the same vein of our "English Majors and women named Debbie" poll a few months back, to start an unscientific poll to see which state has the most groups of letterboxers represented here on the lbGLK list. I started reading through the archives and stopped when my eyes started to cross. Since some people box alone, and some people box in pairs, and some people box in groups of unspecified numbers, each hash mark in my "study" represents a letterboxing "group" - ie, although Deedle & I are two people, I counted us as one "group" since we always box together. So initial results are:

Indiana - 2
Ohio - 2
Illinios - 10
Wisconsin - 6
Michigan - 9
Tennessee - 1 (OK, Delirium hasn't moved yet - she's included in the Michigan count)

If you'd like to be included in this very important occupying of my time, send me an email off list (so we don't clutter the list traffic and cause inboxes to implode). I'll post an update - with trailnames, if no one objects - in a few days. doodle_n_deedle @ yahoo.com

Doodle Bug, who found some time to allow for letterboxing this weekend!!
