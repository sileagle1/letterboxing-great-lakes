I have compiled a listing of the first letterboxes in each MI county.
There are a few where I am still researching the plant date and one
county (Mackinac) where I have yet to pinpoint which Mackinac Island
box was actully the first one planted.

While it would have been a challenging personal goal to find and stamp
each of the "first box in the county" letterboxes, some of these are
no longer active boxes so unless you've already been to them, you're
out of luck. But it is still interesting historical information.

I used several old cached versions of letterboxing.org and searched
the message archives at letterbox-usa to compile the data, but it is
certainly possible I missed something. If you feel there is an error
in any of these (and can confirm an earlier plant date), please let me
know. Also if you can supply any of the missing information, please e-
mail me.

The criteria I used is explained in the text on the page (i.e. no
temporary/gathering boxes or mystery boxes without listed county, etc).

http://www.stampinthebox.com/projects/micounties.asp

Note: This information may not precisely coincide with the boxless
counties initiative because there is at least one MI county that had a
box at one time, but is now boxless.
