I wasn't going to say anything when this was just a debate over
semantics, however I don't take too kindly to accusations that I
have "changed the rules" after the start of the game. I guess I
didn't realize that there were extraneous rules I was supposed to be
following in this game that I made up.

While I think the side-wager game was a cool add-on, the fact that a
bunch of folks got together and placed friendly no-stakes wagers on
when the game will end really should not dictate how I orchestrate
the game. I realize that the side-wagers were placed with the
assumption that there would be one item drawn each day, based on the
game overview I had given. However, this really was meant to be an
explanation as to how the game would move along rather than to be
taken literally as "one and ever only one" item each day.

When I originally laid out the game, I made notes of several
different ideas and strategies I could use to change things up if it
got boring or dragged on -- drawing out multiple items, recognizing
intermediary leaders, etc. In the scheme of the Bingo game itself,
nobody would have cared if there were two items chosen on any given
day, or even if there was a day that I selected 10 items, as
everyone would be getting the bonus. So when I wrote up the game
overview it never occurred to me to go into detail about the
variations I might use at some point during the game as nothing I
was considering would effect the fairness or the outcome of the
*Bingo* game. I did not anticipate that I would be held to a
literal interpretation of the overview or that I wouldn't be able to
change things up a little if I did not explicitly spell out that I
intended to do so.

But to avoid further hard feelings or accusations of unfairness I
will go back to drawing out a single item each day until the end of
the game. Sorry for the confusion.

This is a game, it's supposed to be fun!

Let's please move on...
