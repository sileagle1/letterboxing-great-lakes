From: Debbie Kotlarek (kotlarek@wi.rr.com) | Date: 2006-01-27 21:13:46 UTC-06:00
The "Ignore" feature is nice, but I'd really prefer that the clues be
deleted by the placers if the box is no longer active. Why clutter up the
site? (...and for those that say "because it will change my count!", I'd
ask that you think about all the boxes that DON'T show up on LBNA (WOM
boxes, boxes placed just for an event, clues only on a private website,
bonus boxes, etc.) The LBNA "count" is probably not an accurate count
unless you are a very new letterboxer. I really like the LBNA features to
log the box as found, attempted, etc. and like to see the recent activity on
a box, but DON'T like the fact that people leave clues on the site for boxes
that are no longer active.)

Wisconsin Hiker
