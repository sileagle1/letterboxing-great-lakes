SW Ohio report

From: gaboxing123 (wildskiers@hotmail.com) | Date: 2005-12-08 18:59:33 UTC
I just wanted to finally make a brief box report on some boxes that
we went in search of in October. We were visiting Ohio from Georgia
and had some time to box around the Hillsboro area.

Great Serpent Mound: sorry to report the box is almost a complete
loss: box cracked in mulitple places, puddle of water and logbook
dripping. The stamp was OK, though. We no supplies with us to help
it out. I sent a message to the placer (franzsolo).

Serpent Mound Crop Circles: Box in OK shape, but top is a bit
cracked/open but log book & stamp in OK condition. Logbook getting
full.

Serpent Mound Too: Box fine, stamp fine, logbook very full.

Shepherd: Box is fine, but tree is dead.

Lynchburg Covered Bridge: did not find box; bridge restored &
rededicated in 2005.

Wish we'd had more time to box - maybe next visit!
