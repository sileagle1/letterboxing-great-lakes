From: duncan53211 (duncan53211@yahoo.com) | Date: 2008-08-06 13:58:37 UTC
This is my doing. I was going to post something later today, but
Doodle beat me to it.

I should be finishing up putting all the MPHG boxes on AQ this
morning. Those who planted boxes should have access to the AQ
listings as well as me.

If you planted a box, the only change you might want to make is to the
attributes for the box. I've only been putting "kid friendly" and
"compass needed" as a default. Planters know more, so please update
the attributes as you see fit.

I am also indicating on AQ the latest info I have as to the box
status, including whether a box needs to be adopted as "first aid".
Planters and myself should be able to change this information as
information comes in making it much more efficient.

No one should never change the location of the clues (my website) or
the restrictions.

The listings on LbNA are changing over to being linked to the AQ
listing. I will be looking to transfer ownership of the LbNA listings
down the road, if that is even possible.

This is part of a massive overhaul of my listings at both sites if you
haven't noticed, as well as a major update to my webpages. More on
that later.

Dragon
