From: SpringChick (letterbox@comcast.net) | Date: 2009-01-21 06:20:30 UTC-05:00
While you don't have to use the Yahoo email for anything, you do have to have a Yahoo user ID/account to join a Yahoo group -- thus by default you will have a Yahoo email address even if you choose to never use it, since a Yahoo user ID is automatically assigned a Yahoo email address when the user ID is created.

There is a place you can change the email on all of your Yahoo groups rather easily.

1) Set up and verify your new email address on your Yahoo account
2) Go to: http://groups.yahoo.com/mygroups
3) Log into to your Yahoo account if not already logged in
4) Select, Edit My Groups
5) All of the Yahoo groups that you are a member of will be listed
6) Use the drop-downs to change the email account associated with each group (can be different for different groups)... you can also easily change your message delivery options from here, for instance if you like to turn off email when you go on vacation

SpringChick

However, in her case, because she was using SBC/Yahoo and is not going
to be doing so in the future, she will totally lose access to her Yahoo
ID - will not be able to login or anything. While the groups will still
be there, she will be unable to access them at all. SBC and Yahoo have
some sort of squirrelly deal between them that messes their customers
over in this regard when they stop using SBC as their ISP. I had to
help a friend go through this and the only solution, we were told was to
get her a new free Yahoo ID to replace the old one. It would be much,
much, much easier to do this now, while she has access to the old login
- otherwise, she will not be able to change email preferences, add more
groups (or leave them!), etc. A real pain in the keister.
