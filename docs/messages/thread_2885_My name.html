From: SpringChick (letterbox@comcast.net) | Date: 2006-08-05 09:54:11 UTC-04:00
While it is possible to have more than one Yahoo! identity, it is a bit of a
hassle. It is tempting to want to use different Yahoo! ID's for different
groups, that apply to different areas of your life. But you can only be
signed onto one Yahoo! account at a time on a computer and as your Yahoo!
identity can be tied to other things, having to switch between several is a
pain.

Letterbox-usa was my first experience with Yahoo! groups when I began
letterboxing several years ago. To that point I did not have a Yahoo! ID
for anything -- not mail or a Yahoo! home page or anything. Since my
purpose was to join letterbox-usa, I created a Yahoo! name in line with my
LB trail name. Not long after that I came across a few art-related groups I
wanted to join, and it seemed silly to use SpringChick because it was a name
I used only for letterboxing. So I created a new Yahoo! ID and joined the
art groups as artsychick. Around about that time, Flickr (the photosharing
site I use) was purchased by Yahoo! and incorporated your Yahoo! ID into
your sign-in there. Great, I already had a Flickr ID and it was not
SpringChick or artsychick... so I created yet another Yahoo! ID for this so
I could maintain the identity I had initially started at Flickr. I then
joined a couple of photography related Yahoo! groups using that ID.

I quickly found out what a pain it was to have all these different Yahoo
ID's -- I was constantly signing in and out of Yahoo! as one or the other ID
and couldn't have letterboxing groups open in one browser window and an art
group or my Flickr account open in another window. When I would go to "My
Groups" on Yahoo!, I would see only the groups I had joined using one ID
(whichever one I happened to be signed into at that time), rather than being
able to get a nice comprehensive list of all Yahoo! groups I belonged to.

Eventually I gave up on the multiple personalities and just used SpringChick
for all of them, which significantly simplifies the Yahoo! process.

SpringChick

From: SpringChick (letterbox@comcast.net) | Date: 2006-08-05 10:04:40 UTC-04:00
When you tried creating a Yahoo! ID with "MC Bear", did you have the space
in there? I don't think you can have spaces in your Yahoo! ID. Try using
MCBear or maybe MC_Bear. If that still doesn't work, chances are it is
because somebody is already using the identity you are trying to claim. Try
putting a couple numbers behind it -- like MCBear1234 (people often use a
portion of their phone number, address or birthday).

To create a new Yahoo! ID (the ID you will use for groups):

1) Go to http://groups.yahoo.com/
2) If it says, "welcome [your Yahoo! ID]" and displays a list of groups you
belong to, then you are already signed in with a different ID -- Click on
"Sign Out" to exit this ID.
3) You should now see a light green box with a "Sign In" button and right
below that is a link to "sign up"
4) Click on the "sign up" link and follow the instructions to create a new
account -- this will also create a Yahoo! mail account and personalized "My
Yahoo" page.
5) Once you have successfully created the new Yahoo! ID, then you will need
to go to each of the groups you would like to be a member of and join each
group while signed into Yahoo! with the new ID.

SpringChick
