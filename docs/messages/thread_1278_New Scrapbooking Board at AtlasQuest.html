From: SpringChick (springchick@letterbox-mi.com) | Date: 2005-05-04 11:57:03 UTC
I know there a number of people in this area who do their letterboxing
logbook like a scrapbook. Just thought I would mention in case you
are not an AtlasQuest user, that Ryan has added a Scrapbooking Board
where you can exchange ideas with other letterboxer-scrapbookers.

In case you are not familiar, AtlasQuest is an alternate LB clues type
web site where you can create a profile, enter your boxes and clues
(including HH), post finds, etc. There are also a number of subject-
oriented message boards there -- some letterboxing related, some not
so much. Membership is free. Check it out here:

http://www.atlasquest.com/
