Boxacon 2012 - update

From: mkeaims (aimlesst@sbcglobal.net) | Date: 2012-08-04 21:20:05 UTC
Hi all,

So two out of town wedding trips and one grad class later, I am ready to start working further on the event for this fall!! Here is what is set so far.

I have updated the webpage with everyone who has signed up on atlasquest so far. The consuite page has a list of potluck items so far, if you are thinking ahead at all.

The event box page has boxes that are carved so far. If you want to send a box in for the event, that would be fantastic! The fantasy theme is pretty broad! If there is a set started that you want to contribute to, please feel free. Some of the big series are so sprawling, there is plenty of material out there for carving! If you have a box you want me to plant, please get it in the mail by September 1st to give me time to plant, write clues, and get the clue packets printed. Drop me an email if you need my mailing address. I would like to keep temporary boxes out for several weeks to give people time to get out several days, that worked out really well for Boxacon I and I would really like to run this the same way. If you are going to plant your own boxes and want to be added into the clue packet, please get your clues to me early in September so I can get everything compiled!

aimlesst
