As another obsessive planner, I have started my packing - stuff piling up in the dining room. Oil Change on Wednesday afternoon.

I've had the park map and trail descriptions printed for months. I tried to color code the trails, but some aren't really clear online. I'll mark them again at the park.

I have a spreadsheet with all the attendees listed on AQ. Color code the letterboxers I need to exchange with. PT's with notes. Event box listings. Probably won't find time to get the boxes in Mt. Horeb, but I will be prepared.

I don't print my final checklist until Thursday afternoon. People are still coming and going.
