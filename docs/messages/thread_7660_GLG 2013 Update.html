GLG 2013 Update

From: bostontrsts (letterbox_alcalde@yahoo.com) | Date: 2013-02-19 00:57:13 UTC

There are just over 110 days to go untill GLG and as hosts we have been
pretty quiet. Here is an update so you all know what is going on and
how to plan.

If you don't have a campsite at the Dunes, chances are that you will not
be able to get one. As of this morning there were 2 campsites left for
the weekend of June 14 to 16th. I did say that they go fast...for
alternative campsites you can camp in the National Lakeshore
www.nps.gov/indu/planyourvisit/campgrounds.htm
These are1st
come/1st serve sites, no reservations $15/day Credit/Debit Only.
Please review the lodging information pdf for more information and a
listing of some hotels in the area. Also once you know your lodging
information you can edit the Database to add it in for all.

We also are in need of a few more carvers/stamps. Stamps should be
mailed no later than mid May so we have them prior to June 1. Please
limit the size to fit on a 3x5 index card. Please contact us
individually if you intend to carve so we can maintain a master list and
avoid duplication. Also, please indicate if you want your stamp back.

The itenerary has yet to be set in stone, however expect it within the
next few weeks. Also if anyone is interested in hosting a Friday night
event, there is a shelter in the campground that doesn't require
reservations. I can inquire with the park ranger if we can use it that
evening.
