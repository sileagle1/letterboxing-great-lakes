Traditional message view for group messages - your choice

From: azobox (jparkerg@yahoo.com) | Date: 2006-06-07 19:55:30 UTC
If you prefer the message format that formerly existed here (and other
Yahoo! groups) you may choose to resume that view option.
Simply look towards to top of the message board to some tiny type that
says: Edit Membership

Select that option and then scroll down to choose "Traditional" (vs.
New Improved) from message delivery choices.
Make sure you save your changes.

Voila, back to the familiar message view.

If, however, you prefer the new view, you don't need to do anything.
