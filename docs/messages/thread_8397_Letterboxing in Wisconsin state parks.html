I don't know of anyone that has used a form for planting in a WI State Park.  There have been boxes in place in many WI state parks for 12+ years with little complaints.  Several years ago there was a park ranger at Rib Lake that got kinda fussy, but that was the only time I heard about a problem. The "Don't Ask, Don't Tell" approach has worked well up to this point.
