From: Wisconsin Hiker (kotlarek@wi.rr.com) | Date: 2005-08-03 16:22:19 UTC
Hi!

Several other regions in the U.S. have developed their own patch in
addition to using the national LBNA patch and I thought it would be
nice to have one for our region. Spring Chick has OK'd using the
message board graphic as the basis for a patch for the Great Lakes
region and I am pursuing having some made.

I've posted a graphic of the proposed patch in the FILES section of the
message board (didn't have the .jpg available on this computer, so it
is embedded in a MS Word document). I can post it as a .jpg later
today.

I'm trying to determine if there is much interest among the boxers in
the area in having a patch like this. Cost would be approximately $2
per patch.

Wisconsin Hiker
