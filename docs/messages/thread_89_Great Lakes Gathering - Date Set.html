Great Lakes Gathering - Date Set

From: (duncan53211@yahoo.com) | Date: 2004-04-05 12:05:12 UTC
Fellow letterboxers!

The polling is complete, and we now have a date set for our
gathering! The 2004 Great Lakes Gathering will be on June 12, 2004.

You can go to the "official" website at:

http://home.wi.rr.com/dragonlber/GLG.htm

Remember - if you want to make an event box, whether you'll be in
attendence or not, it can be based on any theme you wish. We would
like to have a number of boxes based on the theme "Cooler by the
Lake", however.

Please email me to let me know whether you plan to attend and
whether you will help in any area noted on the website.

I should have a location picked early in Easter week.

Tim
aka Dragon
