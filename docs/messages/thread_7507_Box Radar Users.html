RE: [LbGLK] Box Radar Users

From: Debbie Kotlarek (kotlarek@wi.rr.com) | Date: 2012-07-12 16:33:26 UTC-05:00
We still mainly rely on printing clues, but find the Box Radar app very
useful. I love how it searches both LBNA and AQ for clues and that when we
are first planning a trip and are still at home, I can enter a specific
location (such as a citys/town) that we will be vacationing in and that it
then saves all the clues in an easily accessible database on the phone.
That means we don't necessarily have to count on having service in a remote
area, but could still access the clues previously saved on the phone. The
database is organized by state and then by city, making it very easy to find
clues. Of course if you have service, it will call up the clues for whatever
area you are currently in.



On our recent trip to Idaho, we decided we didn't really need to print out
all the little "drive-by" clues since our focus would be on hiking, BUT if
the weather was bad or for some other reason we needed/wanted to do some of
these boxes, we could just look up the clues on the phone. It came in handy
on our recent trip since we had forgotten to print clues for a box we wanted
to look for.



Another reason we wanted to use it if we printed clues a while ago and are
having trouble finding a box. It allows us to easily check a clue to see if
it has been updated.
