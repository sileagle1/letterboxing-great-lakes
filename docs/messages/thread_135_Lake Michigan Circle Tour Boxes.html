Lake Michigan Circle Tour Boxes

From: Deb (springchick@letterbox-mi.com) | Date: 2004-04-14 16:56:44 UTC
I've heard from several people and I think there is enough interest
to make this a go. So, if you have a box that meets the criteria
that you would like included in the Lake Michigan Circle Tour series
of boxes, please e-mail me (offlist) the following info:

Your name or trail name (this will be listed on the series page)
Your e-mail (for my use only -- will not be on the series page)
The name of the box
Date the box was placed
URL of the clues
City & State of box
Approx. distance this box is from the tour route

Remember, your clues must tell people where to exit the tour route
and where they will be getting back on. If you are unsure of the
criteria for these boxes see the info file in the FILES section.
There is also a link there to the LM Circle Tour page which outlines
the official tour route.
