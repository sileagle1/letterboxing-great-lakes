Here's a thought, but I don't want to organize it since I'm still
trying to get the Grail series done. But I'd help WI!....

New GL project for the winter months-

1) Each state in the GL region chooses a fairy tale to modify so it
has a letterboxing twist. (I would suggest WI NOT choose Alice in
Wonderland...)

2) A boxer in the state agrees to re-write the story keeping in mind
it will be a children's picture book - each page of text would have a
picture with it - including the cover.

3) Once the modified story is complete, the text is posted online and
boxers in the state sign-up to create a stamp image/picture for each
page. All stamps must be of a uniform size. These individuals would
also complete a box for their stamp.

4) Using the uniform stamp size and the text, printable log-booklets
are created. I'm thinking a pdf file that one could print off and put
together to make a book to use in the collection of the stamps.

5) A designated boxer in the state collects all of the boxes and hides
them in one main location in order as a series. Clues and hiding
location should relate to the theme of the book.

6)Finders of the boxes stamp into the log-booklets to create their own
picture books!

7) Picture books are then used for bedtime stories for children of
letterboxers, or for reading time in classrooms, etc.

With the wealth of great writers we have in the GL region (Sara of LL,
Acorn, Deedle, etc.) and all the great artists/carvers as well, I'd
think this would turn out pretty cool!

BTW, this idea came to my mine while my mom and I discussed the
current display of artwork in the fine arts building at Wisconsin
Lutheran College (where the Never-ending Story box is - the box is in
the Library at WLC, the fine arts center is diagonally next door). A
series of woodblock carvings and other art were created for a book and
they are on display. Although the book is not necessarily a
children's book, the art is pretty good. If you're in the area during
the day you might want to stop by at check it out.
