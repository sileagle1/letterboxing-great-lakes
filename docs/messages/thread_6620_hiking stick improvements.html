From: Kathi Krankoski (krank@ameritech.net) | Date: 2010-06-03 15:59:47 UTC-07:00
In light of my past experience at Boxacon, I've made a few improvements to my hiking stick. First, I've added a longer leather strap which will allow me to sling it over my shoulder while stamping in, so I don't have to set it down and forget it. Second, I've tied a red, white, and blue bandanna to it, in case I still set it down, so it is easier to see. But just in case, as a third fail-safe, I'd really appreciate it if the Wormy Wanderers would follow a few hundred yards behind me whenever I'm on the trail. Thanks in advance!
