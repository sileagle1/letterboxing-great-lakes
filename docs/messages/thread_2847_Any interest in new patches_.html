Any interest in new patches?

From: Wisconsin Hiker (kotlarek@wi.rr.com) | Date: 2006-07-29 03:57:03 UTC
Hi!

Some of you that have been around a while know that we had patches
made with our original regional logo. I placed an order for 200
patches last summer and all 200 were gone by this spring.

I held off reordering the patches, knowing that SpringChick would be
incorporating Minnesota into a new design. I'm willingly to order
some new patches, but am wondering how much interest there is. Since
SpringChick has included a title with the logo, we could also
duplicate the same look on the patch. The patch price would probably
be $2 - $2.50 per patch. Our original patches were a 2.5" round
patch. With the new design we could go with either a circle or
square/rectangle. It could be approximately the same size, or larger
if we wanted.

If you'd like to see the original patch, and two possible designs for
a new patch, you can look in the Photos section of this group, in
the "Patches" album.
http://ph.groups.yahoo.com/group/LbGLK/photos/browse/54cf

I'll wait to see how the discussion goes on this topic, and if there
is enough interest, I'll create a poll to select a final design.

Wisconsin Hiker
